# 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

# 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.

# 출력
첫째 줄에 N!을 출력한다.

=======================================================================================
# 풀이
```cpp
int factorial(int num)
{    
    if (num <= 1) return 1;     
    return num * factorial(num - 1);
}
```
